# Pasta_Java_Exercicios

Repositório de estudos e exercícios de Java, desenvolvido durante o aprendizado da linguagem seguindo a metodologia do curso DevSuperior (Nélio Alves).

## 🎯 Objetivo

Praticar fundamentos de Java (entrada de dados, estruturas condicionais, laços de repetição, strings e arrays) como parte da preparação para atuação como consultor PJ no mercado bancário/corporativo.

## 🛠️ Tecnologias

- **Java** (JDK 25 - Zulu)
- **IntelliJ IDEA**

## 📁 Organização

Os exercícios estão no pacote `exercicios`, com uma classe por exercício:

```
exercicios/
├── Ex1MediaDoisNumeros.java
├── Ex5CelsiusFahrenheit.java
├── Ex6VogalOuConsoante.java
├── Ex7ContandoVogaisEConsoantes.java
└── ...
```

## 🚀 Como executar

Cada exercício possui seu próprio método `main` e pode ser executado individualmente pela IDE ou via linha de comando:

```bash
javac -d out src/exercicios/NomeDoArquivo.java
java -cp out exercicios.NomeDoArquivo
```
